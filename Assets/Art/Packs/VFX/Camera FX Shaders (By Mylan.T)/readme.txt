Asset Documentation for [Camera FX Shaders (By Mylan.T)]

Read documentation online : https://sites.google.com/view/camerafxshaders-packbymylant/home

{About}
This asset allows you to add special effects to the camera, such as blur, glitch, retro, and other visual effects. 

{How to Use}
	
To add the desired effect, first select a GameObject that has a Camera component. 
Then, add the C# script with the effect you want. Set your preferred values and hit Play. That's it! 

{Features}

-Works in both 3D and 2D projects
-Unlike Post-Processing Stack, it has virtually no FPS impact
-Effects are applied only to the camera’s render output (UI/Canvas  remain unaffected)
-You can enable individual effects or layer multiple effects simultaneously
